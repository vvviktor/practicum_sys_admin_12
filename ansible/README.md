### Описание плейбуков:

- project_essentials.yaml - основной плейбук, настройка всей инфраструктуры
- db_clean_init.yaml - загрузка начального состояния БД. Нужен файл my_wiki.sql в папке с плейбуком
- replica_promote_and_switch.yaml - подмена основной базы ее репликой с переводом реплики в основной режим
- img_folder_recovery.yaml - восстановление папки пользовательских изображений. Принимает аргумент -e "backup_name=<имя бэкапа в папке бэкапа изображений на бэкап-сервере>"
- db_restore.yaml - восстановление БД из дампа в папке бэкапа дампов на бэкап-сервере. Принимает аргумент -e "backup_name=<имя бэкапа>
